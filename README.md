# 🔥 EmojExp ️
✨ Emoji-based Regular Expressions ✨

## Example
**Regular Expression** ☠️
```js
new RegExp('^.*(?=,)')
```

**Emoji Expression** 🔥
```js
new EmojExp('🏁✨❤️(👉,)')
```
